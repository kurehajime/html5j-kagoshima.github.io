概要

鹿児島近隣地域のデザイナー・技術者の技術力向上

地方にいてはなかなか得ることが難しい最新の技術に触れる場を作ることで、デザイナー・技術者の学習ニーズを満たし、エンジニアに対して今後どういった学を行えばよいかの指針にする。

鹿児島近隣の産業の活性化

IT企業のデザイナー・技術者の技術向上を行うことで産業の活性化・起爆剤になり得ます。しかしそれ以上に、鹿児島で活動されている企業においても、HTML5という最先端技術に対し積極的に係ることで、様々な産業への技術アピール、企業イメージなどをアピールすることができ、今後の仕事につながります。HTML5の技術を自社の事業に活かしていきたいユーザー企業にとっても、今後の事業展開へのアイデアを得るヒントとなります。

鹿児島近隣学生の技術力向上

地元の学生にとって、都市圏で開催されているカンファレンスに参加することは（特に金銭面において）社会人以上に困難です。彼らの学習ニーズを満たすことで、学生の夢を与えることができ今後の技術力向上につなげることができます。また、このイベントを通じ地元の企業との交流を行うことで地元への雇用へとつなげていきます。

地域コミュニティの活性化

かねてよりデザイナー・技術者の技術力向上に貢献している地域コミュニティと連携をはかることで、コミュニテイをよりいっそう活性化させます。活性化により、より多くの情報を交換することができることで、永続的にデザイナー・技術者のスキルアップを行うことが可能です。

スピーカー

吉川 徹 html5j代表

html5j 代表、HTML5 Experts.jp エキスパート兼副編集長。フリーランスのWebエンジニアとして活動中。 コミュニティ活動では、html5jにてHTML5の最新動向を追うとともにHTML5の普及促進に務める。 また、Google Developer Expert（Chrome）として、Chrome関連の情報発信も行なっている。

浅井 智也 Mozilla Japan

学生時代に Web 技術文書を翻訳、Firefox 1.0 から Mozilla 製品の日本語リリースを担当。インターネットをすべての人々の利益になる方向へ発展させる Mozilla の目標に共感し、大学院卒業後 Mozilla Japan に参加。オープンプロセスに基づく新しいものづくりを日本に広げるべく、Firefox OS の国内展開や Web プラットフォームとアプリのエコシステム活性化に努めるほか、夢を見せるエバンジェリストとして、Mozilla 製品や最新 Web 技術の魅力を伝えている。

物江 修 日本マイクロソフト株式会社

2000年よりマイクロソフト、NTTドコモのジョイントベンチャー立ち上げに参加。技術者として.NET Framework を使用した商用ASPサービスの設計と開発に従事。2004年、マイクロソフト・アジアリミテッド入社。インターネット開発製品のサポートエンジニアを経て現職のデベロッパー＆プラットフォーム統括本部エバンジェリストとなる。以来、ASP.NET、IIS（Webサーバー）の啓発活動に従事し、現在はInternet Explorer、Windows 8アプリを広く開発者に知ってもらうべく活動中。

小松 健作

1972年生まれ。山形県出身。大学時代、家系ラーメン店でのバイトを契機をとして、ラーメン中毒となる。その後、プロトコル研究・CDNシステム開発・VoIP開発など主にインフラ系のエンジニアを経るも、HTML5に出会うことで Web がプラットフォーム化することを確信。2009年よりWebとNWとの相互作用を中心に、HTML5の研究開発に従事。W3Cでの標準化推進や執筆活動、コミュニティ運営による啓蒙など活動は多岐に渡る。Google Developer Expert (HTML5)

秋葉 秀樹（あきば・ひでき）
株式会社ツクロア（tuqulore） 代表取締役・デザイナー

アプリケーションデザインを得意とする株式会社ツクロアCEO・デザイナー、筑波大学非常勤講師。

自身で開発したプロジェクトにより水族館や科学館などの集客にも貢献し、某大手銀行の公式アプリや国産スマートフォンの工場出荷時アプリをデザイン、スタートアップ企業のサービス改善のためのファシリテーターとしてなど、幅広い実績を持つ。最近はサービスデザインを中心に活動、技術も大事だが人と楽しさを重んじるのがモットー。
